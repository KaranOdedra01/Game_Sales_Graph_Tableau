--> 🎮 Video Game Sales Analysis using Tableau

--> Overview

This project is based on analyzing a Video Game Sales dataset using Tableau. The main purpose of this project is to understand sales trends in the gaming industry and visualize the data in a simple and interactive way.

The dashboard helps in identifying:

* Which game genres are most popular
* Which platforms generated higher sales
* Top publishers in the gaming market
* Regional sales comparison
* Year-wise growth in game sales

---

--> Tools Used

* Tableau
* CSV Dataset
* Data Visualization Techniques

---

--> Dashboard Features

* Interactive charts and filters
* Sales comparison by region
* Genre and platform analysis
* Global sales insights
* Easy-to-understand visual representation

---

-> Dataset Details

The dataset contains information related to:

* Game Name
* Platform
* Genre
* Publisher
* Release Year
* Regional Sales
* Global Sales

---

--> Objective

The objective of this project is to perform data analysis using Tableau and convert raw data into meaningful visual insights. This project also helped in improving dashboard designing and data visualization skills.

---

--> Conclusion

Through this analysis, it becomes easier to understand gaming industry trends, customer preferences, and sales performance across different regions and platforms. Tableau helped in presenting the data in a more interactive and understandable format.
